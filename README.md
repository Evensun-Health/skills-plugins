## What is a Claude Code skill?

A skill is a markdown file (or directory of files) that gives Claude persistent, structured knowledge about a domain. When referenced in your Claude Code settings, the skill is automatically loaded when relevant — Claude uses it to answer questions, run workflows, and execute scripts without you having to re-explain the context.

See the [Claude Code documentation](https://docs.anthropic.com/claude-code) for details on installing and authoring skills.
